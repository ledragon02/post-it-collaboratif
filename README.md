# Post-it Collaboratif 

Un tableau de post-its en temps réel utilisant **Node.js** et **WebSockets**.

##  Installation
1. Cloner le projet
2. Installer les dépendances : `npm install`
3. Lancer le serveur : `node server.js`
4. Ouvrir `index.html` dans plusieurs navigateurs.

##  Fonctionnalités
- Création de notes avec ID unique.
- Déplacement synchronisé en temps réel.
- Édition de texte collaborative.
